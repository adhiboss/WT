XP System

Lv.1 Unlock

Earn XP for every task. On-time = bonus XP. Early submission = double XP. Even late submissions earn some points — every bit counts.

🎯 Early: ×2 XP  ·  On time: ×1 XP  ·  Late: ×0.5 XP
🔥

Daily Streaks

Lv.1 Unlock

Complete at least one task every day to grow your streak. Your streak is displayed to the whole class — so losing it hurts. That's the point.

❤️

Hearts System

Lv.2 Unlock

You start with 5 hearts. Miss a task and you lose one. Run out and Duo gets really sad. Refill by completing bonus challenges or waiting 24h.

🏆

Leaderboard

Lv.3 Unlock

Real-time class XP ranking. Everyone sees everyone's score. Social pressure is a proven motivator. Where do you stand?
